Sf4SecurityBundle
==================================

Installation
------------

Step 1: Download Sf4SecurityBundle using composer
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

.. code-block:: bash
    $ composer require sf4/security-bundle

Step 2: Enable the bundle
~~~~~~~~~~~~~~~~~~~~~~~~~

.. code-block:: Markdown
    // config/bundles.php
    //..
    Sf4\SecurityBundle\Sf4SecurityBundle::class => ['all' => true],
    /..
